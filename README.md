# Calci

This project is a simple web-based calculator that allows users to perform basic arithmetic operations such as addition, subtraction, multiplication, and division. It is built using Go for the backend and HTML/CSS for the frontend, providing a responsive and user-friendly interface.

## Features
- Perform addition, subtraction, multiplication, and division.
- Handles invalid input and division by zero gracefully.
- Responsive and clean user interface.
- Built using Go for the server-side logic and HTML/CSS for the frontend.

## Tech Stack
- **Backend:** Go (Golang)
- **Frontend:** HTML, CSS
- **Server:** HTTP server using Go's `net/http` package

1. Install [Go](https://golang.org/) on your system.
2. Clone the repository:
3. Navigate to the project directory:
4. Run the Go server:
go run main.go
5. Open your web browser and navigate to `http://localhost:8080`.

## Usage
1. Enter two numbers in the input fields.
2. Select an operation (Add, Subtract, Multiply, Divide) from the dropdown.
3. Click the "Calculate" button to see the result displayed on the page.
